Studiemateriaal

Alle studiemateriaal wordt online beschikbaar gesteld.

  • Code zal na elke les beschikbaar gesteld worden

  • Deze online cursus is altijd integraal beschikbaar.

Het wordt sterk aangemoedigd de online documentatie van Microsoft te raadplegen om zich te verdiepen.

Aanpak

Lessen worden gegeven als een mix van theorie en live coding tutorial. Alle code uit de les komt integraal online. Jouw lesgever zal aangeven waar je deze kan vinden.

Je wordt wel verondersteld mee te denken, maar niet verondersteld mee te volgen op je eigen laptop.

Er worden oefeningen beschikbaar gesteld voor studenten die materie willen inoefenen.
