SQL Server Docker

Deze sectie is nog niet volledig uitgeschreven, maar bevat alle notities om de implementatie te doen.

Waarom?

  • Free limit bereikt op Azure → Alternatief nodig

  • Lokaal SQL server testen zonder impact op OS/installatie.

  • Er bestaat geen native SQL Server voor macOS, wel voor Windows en Linux.

Installeren

docker pull mcr.microsoft.com/mssql/server:2022-latest
docker run -e "ACCEPT_EULA=Y" -e "MSSQL_SA_PASSWORD=Wachtw00rd1" `
-p 1433:1433 --name sql1 --hostname sql1 `
-d `
mcr.microsoft.com/mssql/server:2022-latest

De server admin username is sa.

ShipIt DB maken

connect (met VS Code ofzo)

Op master DB
CREATE DATABASE [shipit-identity]
go

CREATE LOGIN IdentityServer WITH PASSWORD = 'WachtW00rd'
go
Op shipit-identity DB
CREATE USER IdentityServer FOR LOGIN IdentityServer;
GO
GRANT CREATE TABLE TO IdentityServer
GO

CREATE SCHEMA pg3is
GO
GRANT ALTER ON SCHEMA :: pg3is TO IdentityServer
GO
GRANT INSERT, UPDATE, DELETE, SELECT, REFERENCES ON SCHEMA :: pg3is TO IdentityServer
GO

CREATE SCHEMA pg3users
GO
GRANT ALTER ON SCHEMA :: pg3users TO IdentityServer
GO
GRANT INSERT, UPDATE, DELETE, SELECT, REFERENCES ON SCHEMA :: pg3users TO IdentityServer
GO

Migrations

in IS project

connectionstring
Server=tcp:localhost,1433;Initial Catalog=shipit-identity;Persist Security Info=False;User ID=IdentityServer;Password=WachtW00rd;MultipleActiveResultSets=False;Encrypt=Optional;TrustServerCertificate=True;Connection Timeout=30;
Migrations
dotnet ef database update -c ConfigurationDbContext
dotnet ef database update -c UserDbContext
