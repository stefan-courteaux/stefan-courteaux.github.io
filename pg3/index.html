Welkom bij Programmeren Gevorderd 3!

Situering

  • Programmeren Gevorderd 2 behandelt implementatiedetails van een enkelvoudige backend.

  • Programmeren Gevorderd 3 behandelt de architecturale complexiteit van een cloud-hosted backend met meerdere services en de basisintegratie van een zeer eenvoudige frontend.

  • Web 3 behandelt het bouwen van een volwaardige moderne Single Page Web App, met nadruk op de frontend.

Inhoud

  • Iteratief bouwen van een .Net webapi systeem

    • Microservices

    • Azure SQL, Storage, Cosmos, Keyvault…​

  • Verkennen van technologie

    • Docker

    • Identity Server…​

  • Azure DevOps development flow leren kennen.

  • Systeem installeren in de Azure cloud.

Tooling

Microsoft investeert sinds 2014 enorm in een platformonafhankelijk .Net ecosysteem dat werkt op Windows, macOS en Linux.

In de serverwereld is Linux dominant, ook bij Microsoft.

Waarom geen klassieke Visual Studio?

Deze moderne tools verschijnen steeds vaker in het werkveld. Ze leren gebruiken is belangrijk om jouw kansen op de arbeidsmarkt te verhogen.

Is dat dan verplicht?

Neen. Ik doe geen enkele moeite om dat te controleren of af te dwingen.

Aanpak

De rode draad doorheen de lessen is de ontwikkeling van een klein systeem voor een bedrijf dat pakjes levert, zoals PostNL of UPS.

De nadruk ligt op het leggen van architecturale puzzel, niet op (business) logica.

Gestaag zullen er nieuwe business en technische requirements aangebracht worden. De implementatie zal besproken en getoond worden tijdens de les. Het eindresultaat is een cloud applicatie met beknopte functionaliteit.

Broncode In Deze Cursus

Tijdens de lessen wordt er aan live coding gedaan. Dit heeft als gevolg dat de code uit de les, en dus in het code repository, niet identiek zal zijn aan deze in de cursus.

Laat je hierdoor niet in de war brengen. Je zal snel ondervinden dat deze afwijkingen zeer oppervlakkig zijn en het geheel conceptueel identiek is aan de cursus. Immers: de lessen volgen de cursus, niet omgekeerd.
