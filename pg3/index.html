Welkom bij Programmeren Gevorderd 3!

Doel

  • Iteratief bouwen van een data-driven async .Net webapi systeem.

  • Azure DevOps development flow leren kennen.

  • Systeem installeren in de Azure cloud.

Aanpak

De rode draad doorheen de lessen is de ontwikkeling van een klein systeem voor een bedrijf dat pakjes levert, zoals PostNL of UPS.

Bouw in parallel met de lessen mee aan dezelfde app om het materiaal in te oefenen voor je aan je eigen project begint.

Gestaag zullen er nieuwe business en technische requirements aangebracht worden. De implementatie zal besproken en getoond worden tijdens de les. Het eindresultaat is een cloud applicatie met beknopte functionaliteit.

Evaluatie

De student kiest een onderwerp waarrond deze individueel een cloud project zal bouwen en deployen. Geen pakjesbedrijf of architecturale copy-paste van het project uit de lessen, maar iets grofweg equivalent qua complexiteit.

Consulteer de Studiewijzer voor meer informatie betreffende de evaluatie.

Waarom geen klassieke Visual Studio?

Deze moderne tools verschijnen steeds vaker in het werkveld. Ze leren gebruiken is belangrijk om de kansen van de student op de arbeidsmarkt te verhogen.
